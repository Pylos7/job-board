# Django Models + Jobs Board Project

This GitHub project documents the development of a Jobs Board using Django Models. Below are the steps taken in the project:

- Introduction to Django Models.
- Setting up the project and application.
- Designing the model for job postings.
- Incorporating initial job postings into the database.
- Extending the job posting model with a new field.
- Retrieving active job postings within the view.
- Displaying active jobs in the template.
- Implementing filters for job listings formatting.
- Creating detail pages for job postings.
- Managing error handling for job detail pages.
- Adding styling to home and job detail pages.

This project provides a detailed walkthrough of building a practical Jobs Board application using Django Models.
